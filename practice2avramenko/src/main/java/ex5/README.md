## **Завдання №5 - Обробка колекцій  (28.03.25)**
___
### :scroll: **Завдання**

>1.  Реалізувати можливість скасування (undo) операцій (команд).
>
>2. Продемонструвати поняття "макрокоманда"
>

>
>4. Забезпечити діалоговий інтерфейс із користувачем.
>
